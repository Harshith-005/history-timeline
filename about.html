About This Project

A lightweight educational site to learn history and practice modern Git workflows.

Project goals

Provide a clean, responsive timeline that is easy to extend. Use this as a demo to practice branching, commits, PRs and deployment.

Contribute

  1. Fork the repository on GitHub
  2. Create a branch for your change (e.g., feature/add-event)
  3. Add content and open a Pull Request
