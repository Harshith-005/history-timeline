About this project

This simple static site was built as a learning project: to practice Git workflows (branches, commits, merges) while creating a clean, responsive timeline presentation. You can extend it by adding more events, images, and interactivity.

How to contribute

  • Fork the repo on GitHub
  • Create a branch for your changes
  • Add or improve event pages
  • Open a Pull Request
